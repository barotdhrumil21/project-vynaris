08 · Do / don't

The rules that keep this instrument sharp.

doaccent discipline
blocking · you
Cobalt only when we need you to act.

Accent is a light, not a decoration. If it's on, a human must do something.

don'taccent bloat
new hot pro
Everything is urgent → nothing is urgent.

Never use cobalt for decoration, tags, navigation, or secondary CTAs. Use ink or neutral instead.

donumerals with weight
3.9 / 3.2%
Tnum grotesque, negative tracking, on a tick rule.

Numbers earn size. The target sits quietly in ink-3 so the value dominates.

domono for structure
g-0142 · Q3 · wk 06 · ck_8fA2xR
IDs, timestamps, run_ids, tool names.

Monospace carries the infra voice. Use it for anything a machine would recognize.

don'tdecorative type
Friendly greetings! 👋 Welcome to the dashboard!
Grotesk is for structure, not warmth.

Space Grotesk is for headlines, KRs, section titles. Don't use it for body prose or chatty copy.

dorules > shadows
Elevation by hairline.
Only dialogs get shadows. Everything else is lines.

Hierarchy comes from line weight and contrast, not drop shadows. Keeps the system feeling drafted, not cushioned.

Signature moves
Display face. Space Grotesk for headlines & KR numerals — engineered character without being quirky.
Mono structure. IDs, timestamps, tool names, budgets — monospace carries the "infra" feel.
Cobalt accent. #1F4FD9 — single saturated hue for decisions & links. Green/amber/red for status only.
Earned density. Tool trace with latency + cost. Policy diff inline. Budget meter. Real artifacts, not decoration.
