07 · Do / don't

The rules that keep this instrument sharp.

doaccent discipline
blocking · you
Persimmon only when we need you to act.

Accent is a light, not a decoration. If it's on, a human must do something.

don'taccent bloat
new hot pro
Everything is urgent → nothing is urgent.

Never use persimmon for decoration, tags, navigation, or primary CTAs. Use ink instead.

donumerals with weight
3.9 / 3.2%
Tnum grotesque, negative tracking, the number is the hero.

Numbers earn size. The target sits quietly in ink-3 so the actual value dominates.
